## Overview
This project reimagines FedEx's Debt Collection Agency (DCA) management using AI-driven decision intelligence.  
Instead of static reporting, the system provides real-time recovery prediction, profit optimization, and governance recommendations.

## Key Features
- Logistic Regression based recovery prediction (explainable AI)
- Real-time case analysis and decision recommendation
- Profit and ROI-aware recovery prioritization
- Counterfactual "What-If" decision simulation
- DCA performance intelligence
- Governance and write-off risk monitoring
- Executive dashboard with live KPIs

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- Plotly
